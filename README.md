
# Whale

<!-- badges: start -->
<!-- badges: end -->

Say a phrase with a whale on the screen.

## Installation

You can install the development version of Whale from [GitHub](https://github.com/) with:

``` r
# install.packages("devtools")
devtools::install_github("harrisonpapa/Whale")
```

## Example

``` r
library(Whale)
## basic example code
say("Hello World!")
```
